# 智能文本处理工具 - 广东东软学院首届AI编程黑客马拉松

## 题目描述
使用Python开发一个智能文本处理工具，能够对用户输入的文本进行多种分析和处理操作。本题重点考察基础编程能力和字符串处理能力。

## 功能要求
实现一个 `TextProcessor` 类，包含以下方法：
1. `word_count(text)` - 统计文本中的单词数量（中英文混合文本）
2. `char_frequency(text)` - 统计每个字符出现的频率，返回前5个高频字符
3. `sentiment_analysis(text)` - 简单的情感分析（基于关键词匹配）
4. `text_summary(text, n=3)` - 提取文本摘要（取前n个句子）
